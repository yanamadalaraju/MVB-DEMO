

# MV Branding Strategy — Premium Corporate Website

## Design System Setup
- Custom color palette: Blue (#1E3A8A → #3B82F6), Magenta (#A21CAF → #E879F9), Green (#047857 → #34D399)
- Animated gradient CSS variables blending Blue → Magenta → Green
- Glassmorphism utility classes (backdrop-blur, translucent backgrounds, soft shadows)
- Typography: Inter font family
- Dark mode support with theme toggle
- Animated mesh gradient backgrounds and floating particle effects using pure CSS/canvas

## Global Layout & Navigation
- Sticky transparent header with blur effect, logo, nav links, dark mode toggle, and "Get Consultation" CTA
- Animated hamburger menu for mobile with slide-in drawer
- Scroll progress indicator bar at top
- Smooth scroll navigation between sections
- Loading splash screen animation on first visit
- Elegant animated footer with gradient accents, social links, and company info

## Page 1: Home (/)
- **Hero Section**: Large headline "Customer Centricity with Peace of Mind", subheadline about Data Cabling & Security, two CTA buttons with glow effects, animated network-line background with gradient mesh
- **About Snapshot**: Brief company intro with fade-in animation
- **Core Values Preview**: Horizontal scroll or grid of value icons with hover glow
- **Services Preview**: 3-4 animated cards with icons, hover tilt, and gradient borders
- **Why Choose Us**: Stats section with animated counters (Projects Delivered, Clients Served, Years of Experience)
- **Call to Action**: Full-width gradient banner with consultation CTA

## Page 2: About Us (/about)
- Vision & Mission sections with scroll-triggered fade-in animations
- "Our Approach" section: 3-step animated process flow (Right Products → Right Solutions → Right Value Proposition)
- Animated timeline or vertical stepper showing company journey

## Page 3: Services (/services)
- Premium service cards for: Data Cabling Infrastructure, Data Security Solutions, End-to-End IT Deployment, Strategic IT Core Development
- Each card: icon, hover glow animation, subtle 3D tilt effect on hover, gradient border, "Learn More" button
- Expandable detail sections or modal for each service

## Page 4: Core Values (/values)
- Animated grid of 8 value cards with glowing hover effects
- Values: Customer Centricity, Peace of Mind, Innovative Solutions, Value for Talents, Continuous Learning, Accountability, Commitment, Future Ready, Result Oriented
- Scroll-triggered staggered fade-in animations

## Page 5: Brand Philosophy (/brand)
- Three color psychology sections (Blue, Magenta, Green) that animate into view on scroll
- Each section uses its respective color as dominant, with visual examples of how it's used in the brand
- Dynamic color transitions between sections

## Page 6: Contact (/contact)
- Contact form with Name, Email, Company, Message fields and validation
- Consultation booking CTA section
- Animated gradient background
- Company contact details and social media icons
- Success animation on form submission

## Premium Effects (Applied Throughout)
- Animated gradient blobs using CSS keyframe animations
- Network-line/particle canvas background on hero sections
- Interactive button glow and hover micro-interactions
- Scroll-triggered fade-in, slide-up, and scale animations using Intersection Observer
- Glassmorphism cards and sections throughout
- Smooth page transitions between routes
